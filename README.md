# sgol-server

# Description

**sgol-server** is a server implementing the Spatial Graph Operations Language.

# Building

```
cd scripts
bash build.sh
```

# Usage

You need to set 2 environment variables to use **sgol-server**.

```
export SGOL_CONFIG_PATH=/path/to/sgol.hcl
```

```
./sgol-server serve
```
